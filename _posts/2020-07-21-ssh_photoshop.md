save for web with quality %50
```
osascript -e '
activate application "Adobe Photoshop 2020"
tell application "System Events" to keystroke "s" using {shift down, command down, option down}
tell application "System Events" to key code 48
tell application "System Events" to keystroke "50"
'
```

new file 1920x1080 px
```
osascript -e '
activate application "Adobe Photoshop 2020"
tell application "System Events" to keystroke "n" using command down
tell application "System Events" to key code 48
tell application "System Events" to keystroke "1920"
tell application "System Events" to key code 48
tell application "System Events" to keystroke "1080"
tell application "System Events" to key code 36
'
```
